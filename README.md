# PV_array_computer_vision

1. Put "nr3.h", "ludcmp.h", "svd.h" given by "W.H.Press, et. al.: Numerical Recipes, 3rd ed." in the same directory as above code files
2. Create "Data" directory 
3. Put image coordinate files like "IMG_0752array_A.txt" in the Data directory
4. g++ main.cpp (compile)
5. ./a.out (run)
